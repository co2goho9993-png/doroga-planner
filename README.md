# Doroga 2026 Planner

Веб-приложение для редактирования общего планнера выставки «ДОРОГА 2026» в формате A3 landscape с экспортом в PDF A3 400 dpi.

## Запуск

Нужен Node.js 18 или новее.

```bash
npm start
```

По умолчанию приложение откроется на:

```text
http://127.0.0.1:5196/
```

Можно задать другой порт:

```bash
PORT=3000 npm start
```

В PowerShell:

```powershell
$env:PORT = "3000"; npm start
```

## Деплой на Vercel

При деплое укажи корневой папкой проекта именно папку с этим файлом:

```text
doroga-planner
```

В проекте есть `vercel.json`: он добавляет заголовки для локальных шрифтов и vendor-файлов. Шрифты должны лежать в `fonts/` и быть закоммичены в GitHub.

## Проверка перед коммитом

```bash
npm run check
```

Скрипт проверяет, что основной HTML читается, inline JavaScript синтаксически корректен, а локальные шрифты и vendor-библиотеки существуют.

## Структура

```text
index.html       Основное приложение
server.cjs       Локальный статический сервер
fonts/           Локальные шрифты макета
vendor/          Локальные html2canvas и jsPDF
scripts/check.cjs Проверка проекта
```

## Важное

Шрифты лежат в репозитории как локальные ассеты проекта. Перед публикацией в публичный GitHub-репозиторий проверь права на распространение этих файлов.

Сохранение состояния делается кнопкой `Сохранить` в JSON-файл. Загрузка состояния — кнопкой `Загрузить`. Экспорт PDF выполняется кнопкой `PDF A3 400dpi`.
